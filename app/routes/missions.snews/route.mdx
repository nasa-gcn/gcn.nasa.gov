[SNEWS](https://snews2.org/) is a consortium of MeV neutrino facilities which work together to distribute early warning alerts for core-collapse supernovae, reporting the neutrinos released during the collapse event before the first light of the supernova escapes.

**JSON-Serialized SNEWS2 Notices in GCN Kafka:**

SNEWS2 distributes notices for neutrino coincidence alerts generated by the SuperNova Early Warning System (SNEWS 2.0) in the JSON format using GCN Kafka.

The SNEWS 2.0 coincidence network provides early alerts for candidate core-collapse supernovae by identifying statistically significant neutrino coincidences across participating neutrino observatories, including Super-Kamiokande, IceCube, and KM3NeT. Notices are issued for both real and test alerts. Real alerts may include triangulation information when available to improve source localization.

For schemas and examples see [GCN Schema](/schema/stable/gcn/notices/snews2) or use the [Schema Browser](/docs/schema/stable/gcn/notices/snews2/).

Detailed descriptions and examples of SNEWS2 Notices are available in the [GCN Schema GitHub project](https://github.com/nasa-gcn/gcn-schema/tree/main/gcn/notices/snews2).
